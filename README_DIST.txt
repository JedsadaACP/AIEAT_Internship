AIEAT News Analyzer (Internship Project 2026)
==============================================

Installation
------------
1. Extract the zip folder.
2. Ensure the 'data' folder exists next to the .exe (it contains 'schema.sql').
3. Run 'AIEAT_News_Analyzer.exe'.

Setup
-----
- The first run will create 'data/aieat_news.db'.
- To enable AI features, ensure Ollama is running.
- Default settings have been applied.

Troubleshooting
---------------
- If the app opens and closes immediately, check 'logs/app.log'.
- Ensure you have write permissions to the 'data' folder.
