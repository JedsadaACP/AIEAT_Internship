AIEAT - AI-Enhanced Article Tracker v1.0.0
Thank you for installing AIEAT!
Getting Started:
1. Launch AIEAT from your Start Menu or Desktop shortcut
2. Select a profile (Finance, Tech, or Politics)
3. Add news sources and set your keywords
4. Click Scrape to fetch articles
5. Click Batch Score to run AI relevance scoring
Requirements:
- Windows 10 or later
- Ollama AI Engine (can be installed during setup)
For full documentation, see User_Manual.md or visit:
