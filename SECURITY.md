# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in AIEAT, please report it privately by emailing:

**166399126+JedsadaACP@users.noreply.github.com**

Do not open a public GitHub issue for security vulnerabilities.

## Expected Response Time

We will acknowledge your report within **48 hours**. After acknowledgement, we will work with you to understand and address the vulnerability, and provide updates on our progress.

## Scope

This security policy applies to:
- The AIEAT desktop application
- The AIEAT codebase hosted on GitHub
- All official releases and distributions

## Out of Scope

- Social engineering attacks
- Physical security
- Issues related to the user's operating system or hardware

Thank you for helping keep AIEAT secure!
