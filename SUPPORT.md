# Support

## Getting Help

If you need help with AIEAT, here are your options:

### Bug Reports
If you encounter a bug, please open an issue on our [GitHub Issues](https://github.com/JedsadaACP/AIEAT_Internship/issues) page using the **Bug Report** template.

### Feature Requests
Have an idea for improvement? Open an issue using the **Feature Request** template.

### Documentation
- **English User Manual:** [User_Manual.md](./User_Manual.md)
- **Thai User Manual:** [User_Manual_TH.md](./User_Manual_TH.md)
- **Quick Start:** [QuickStart_Guide.md](./QuickStart_Guide.md)

### Application Logs
If the app crashes or behaves unexpectedly, check the log file at:

- **Installed app:** `C:\Program Files\AIEAT\logs\app.log`
- **Development mode:** `<project_root>\logs\app.log`

Include relevant log entries when reporting bugs.

### Security Issues
For security vulnerabilities, please refer to our [Security Policy](./SECURITY.md). Do **not** open a public issue for security problems.
